# Custom LiPD Download

This repository was generated by [PReSto](https://paleopresto.com) (Paleoclimate Reconstruction Storehouse) from the [LiPD Downloads Template](https://github.com/DaveEdge1/lipd-downloads). It downloads LiPD records from [LiPDverse](https://lipdverse.org) over the parameters captured below.

## Proxy data selection

- **Mode:** filtered (records hand-selected via PReSto's interactive map interface and queried from LiPDverse)
- **Archive types requested:** Coral

(See `query_params.json` for the full TSID list.)

## Downloaded data

- **Output:** `lipd_files.zip` — a zip archive of raw `.lpd` files produced by the `lipdGenerator` Docker container (`davidedge/lipd_webapps:lipdGenerator`) from the records listed in `query_params.json`.
- The data and the inputs that produced it are uploaded as the `lipd-download-data-<run_id>` workflow artifact (90-day retention).

## Citation & archive

Each successful download is bundled into a tagged GitHub Release named `download-<run_id>` at <https://github.com/tanaya-g/presto-lipdDownload-17804421724051555/releases>. The release preserves the LiPD data file, `query_params.json`, the cleaning report (if any), and the README snapshot — these survive beyond the 90-day Actions artifact retention so the download remains fully auditable and reproducible long after the workflow logs expire.

If [GitHub-Zenodo integration](https://docs.github.com/en/repositories/archiving-a-github-repository/referencing-and-citing-content) is enabled on this repository, each release also receives a citable DOI, with a stable concept DOI for the download series as a whole.
